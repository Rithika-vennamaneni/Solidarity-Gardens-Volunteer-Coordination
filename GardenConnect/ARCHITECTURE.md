## ğŸ¯ Matching Algorithm Logic

```
Input:
  - Garden with needs
  - List of all volunteers

Process:
  For each volunteer:
    1. Calculate Skills Match
       matchingSkills = volunteer.skills âˆ© garden.skills_needed
       skillScore = (|matchingSkills| / |garden.skills_needed|) Ã— 40

    2. Calculate Days Match
       matchingDays = volunteer.available_days âˆ© garden.days_needed
       dayScore = (|matchingDays| / |garden.days_needed|) Ã— 30

    3. Calculate Times Match
       matchingTimes = volunteer.available_times âˆ© garden.times_needed
       timeScore = (|matchingTimes| / |garden.times_needed|) Ã— 30

    4. Total Score
       totalScore = skillScore + dayScore + timeScore
       matchPercentage = round(totalScore)

    5. Create match object
       {
         volunteer,
         matchPercentage,
         matchingSkills,
         matchingDays,
         matchingTimes
       }

  Sort matches by matchPercentage (descending)

Output:
  Sorted array of matches with scores
```
